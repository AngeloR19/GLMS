# GLMS – Global Logistics Management System

A logistics management system developed using ASP.NET Core MVC, ASP.NET Core Web API, Entity Framework Core, SQL Server, and Docker.

## Features

* Client Management
* Contract Management
* Service Request Management
* Authentication & Authorisation
* RESTful API Integration
* Automated Integration Testing
* Docker Containerisation

## Technologies

* C#
* ASP.NET Core MVC
* ASP.NET Core Web API
* Entity Framework Core
* SQL Server
* Docker
* NUnit

## Architecture

The application follows a Service-Oriented Architecture (SOA) where the MVC frontend communicates with a separate backend Web API using HttpClient. The solution is containerised using Docker and Docker Compose.
